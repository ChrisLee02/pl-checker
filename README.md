> 2017f Programming Language Homework Checker

# HOW TO USE
```bash
git clone git@github.com:Jhuni0123/pl-checker.git
cd pl-checker
cd hw1
# write ex1.ml, ex2.ml, ... ex4.ml
make test 1 # check exercise 1
make        # check every exercise
```
